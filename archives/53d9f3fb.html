4宝塔脚本定时任务无法执行

一次在脚本里写这两句:

git config --global user.name “your name here”

git config --global user.email “your email@example.com”

报错为

fatal: $HOME not set

最终的解决办法为将上述两行改为

git config --system user.email “your name here”
git config --system user.name “your email@example.com”
